Bootstrap
Se trata de um framework para desenvolvimento de sites resposnivos
Possui uma vasta biblioteca de componentes que facilita agiliza o desenvolvimento de página web
É um dos frameworks mais conhecidos e utilizados para desenvolvimento front-end
